# Studio L4

Site institucional estático para desenvolvimento de sites e marketing digital.

## Estrutura

- `index.html`: página principal.
- `styles.css`: estilos responsivos.
- `script.js`: comportamento do menu e cabeçalho.
- `*.png`: ativos visuais da marca.

## Publicação no Git

Este projeto pode ser publicado em GitHub Pages, Netlify, Vercel ou outro host que aceite sites estáticos. A raiz de publicação é o próprio diretório do repositório.
